# Exercícios básicos de JS

Perguntas simples para iniciantes em JS, com o intuito de aprimorar suas práticas 💪🏾🧠


## 1 - Variáveis e Template Strings
- Criar duas variáveis:
    ```
    (nome, idade)
    ```
 
 - Em seguida, exibir uma mensagem com essas informações.

- Saída esperada: Meu nome é Rodrigo e tenho 23 anos.


## 2- Operadores aritméticos

- Criar duas variáveis aritméticas:
    ```
    - let numero1 = 18;
    - let numero2 = 7;
    ```

- Exibir o resultado das seguintes operações:
    - Adição
    - Subtração
    - Multiplicação
    - Divisão
    - Resto da divisão


## 3 - Concatenação de *strings*

-  Criar uma variável:
    ```
    - let preco = 250;;
    ```

- Utilizar concatenação para exibir a seguinte mensagem:
    - O produto custa R$250.


## 4 - Condicional *if/else*

- Criar uma variável:
    ```
    - let idade = 20;
    ```

- Verificar se a pessoa é maior ou menor de idade.

- Saída esperada:
    - Você é maior de idade.
    - Você é menor de idade.

## 5 - Aprovação

- Criar uma variável:
    ```
    let nota  = 7;
    ```

- Se a nota for maior ou igual a 6, informe:
    - Aprovado(a).
- Caso contrário:
    - Reprovado(a).

## 6 - Verificação de senha

- Criar uma variável:
    ```
    - let senha = "1234";
    ```

- Verificar se é igual a *"admin"*.

- Exiba:
    - "Acesso permitido."
    - "Acesso negado:"

## 7 - Operador condicional ternário

- Reescrever o exercício da maioridade utilizando o operador condicional ternário.

## 8 - Temperatura

- Criar uma variável:
    ```
    - let temperatura = 32;
    ```

- Utilizar o operador ternário para informar:
    - "Muito quente"
    - "Temperatura agradável"

## 9 - Saldo bancário

- Criar uma variável:
    ```
    - let saldo = 150;
    ```

- Utilizar o operador ternário para verificar se o saldo é positivo.


## 10 - Função com parâmetro

- Criar uma função chamada:
    ```
    - cumprimentar(nome);
    ```

- Deverá retornar uma saudação personalizada.

## 11 - Função de soma

- Criar uma função:
    ```
    - somar (a,b);
    ```

Deverá retornar a soma dos dois números.

# 12 - Par ou ímpar

- criar uma função:
    ```
    - parOuImpar(numero);
    ```

- Deverá informar se o número fornecido é par ou ímpar.

## 13 - Parâmetro com valor padrão

- Criar uma função:
    ```
    - boasVindas(nome = "Visitante");
    ```

- Caso nenhum argumento seja informado, utilizar o valor padrão "Visitante".

## 14 - Cálculo de média (Desafio)
- Criar uma função:
    ```
    - calcularMedia(n1,n2);
    ```
- Deverá retornar:
    - Calcular média das duas notas;
    - Considerar aprovado quem tiver nota maior ou igual a 7;
    - Usar operador ternário para verificar se o aluno foi aprovado;
    - Retornar a média e a situação do aluno.
